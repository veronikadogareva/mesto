# Название проекта
Mesto Russia
## Описание проекта
Проект представляет собой страницу профиля, который можно редактировать. А именно: 
*добавлять, удалять и отмечать понравившиеся места;
*редактировать профиль;
## Используемые технологии
*HTML5  
*CSS
*JavaScript
*Webpack
